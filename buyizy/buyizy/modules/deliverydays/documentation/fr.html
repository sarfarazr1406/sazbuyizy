Présentation

Ce module permet au client de choisir un jour de livraison.

Vous pouvez exporter les commandes à livrer et les produits à préparer pour une date précise.

Si vous le voulez la date de livraison demandée peut apparaître sur la facture et le bon de livraison. Dans ce cas une modification des fichiers de Prestashop est nécessaire.

Installation du module

Installation à partir du panneau d'administration

C'est la méthode la plus simple, mais elle peut ne pas fonctionner dans certains cas. Si cela ne fonctionne pas, utilisez la méthode d'installation manuelle présenté ci-dessous.

Dans l'onglet modules, cliquez sur ajouter un nouveau module.

Cliquez sur Parcourir pour ouvrir la boîte de dialogue qui vous permet de rechercher sur votre ordinateur, sélectionnez le fichier puis validez la boîte de dialogue. Enfin, cliquez sur Télécharger ce module.

Le module apparaîtra alors dans votre liste des modules sous l'onglet Module.

Cliquez sur le bouton Installer pour installer le module.

Installer un module manuellement

Utilisez cette méthode si l'installation manuelle ne fonctionne pas. Elle peut sembler fastidieuse, a priori, pour les débutants, parce qu'elle requiert un certain nombre de connaissances en FTP, etc., mais elle n'est pas plus complexe et peut-être même plus pratique car elle fonctionne à chaque fois.

En outre, le savoir-faire nécessaire à l'installation est assez basique et c'est quelque chose que tous les gestionnaires de boutique doivent maîtriser.

Pour installer manuellement un nouveau module de PrestaShop:

  1. Décompressez le fichier d'archive du module (.zip). Il devrait en résulter un nouveau dossier.
  2. En utilisant de votre client FTP, placez le dossier dans votre dossier PrestaShop /modules. Faites attention à ne pas télécharger ce dossier dans le dossier d'un autre module (ce qui peut arriver lorsque l'on fait glisser des éléments). Téléchargez le dossier en entier, pas seulement les fichiers qu'il contient.
  3. Accédez à votre back-office, dans l'onglet 'Modules'.
  4. Repérez le nouveau module dans la liste des modules. Vous pourriez avoir à faire défiler; vous pouvez également utiliser le moteur de recherche de la liste, qui devrait vous donner une liste mise à jour dynamiquement de la correspondance des noms de modules.
  5. Dans la ligne du nouveau module, cliquez sur le bouton «Installer».

Utilisation

  1. Définissez les jours de livraison possible dans la page de configuration du module
  2. Un calendrier apparait à côté du choix du transporteur lorsque un client passe commande lui permettant de choisir une date de livraison.
  3. La date choisie apparait dans le détail de commande

Paramètres

Gérer par groupe

Vous pouvez définir les jours des livraisons globalement pour tous les clients ou par groupe de clients. Si, par exemple, vos clients sont classés par zone de livraison cela peut être utile.

Activer pour ...

Cette option n'apparait que si "Gérer par groupe" est activé. Choisissez quels groupes de clients peuvent choisir un jour de livraison.

Requis pour commander

Activer cette option pour forcer le client a choisir un jour de livraison. Il ne pourra pas finaliser sa commande sinon.

Envoyer la date de livraison à ...

Quand une commande est créée le module peut envoyer la date de livraison demandée par e-mail à un employé. Sélectionnez "Personne" pour désactiver cette fonctionnalité.

Envoyer la date de livraison au client

Quand une commande est créée le module peut envoyer la date de livraison demandée au client pour rappel.

Utiliser un e-mail séparé

N'apparait que si "Envoyer la date de livraison au client" est activé. La date de livraison demandée peut être envoyée dans la confirmation de commande ou dans un e-mail séparé.

Cet email peut être modifier dans l'outil de traduction de Prestashop.

Voir la date de livraison dans la liste des commandes

Cette option ajoute une colonne "Date de livraison" dans l'onglet « Commandes »

Jours de livraison

Si l'option "Gérer par groupe" est activée, les valeurs suivantes sont a définir pour chaque groupe.

Livrer

Pour chaque jours de la semaine, choisissez si la livraison est possible

Date limite de livraison

Pour chaque jour livrable vous devez définir le moment à partir duquel il n'est plus possible de commander. Par exemple si pour livrer une commande le mardi elle doit être passé le lundi avant 14h, indiquez en fasse de mardi 1 jour(s) avant à 14:00

Cette option est redondante, mais plus complète, avec l'option "Jours de préparation"

Jours de non livraison

Sélectionnez les jours où la livraison est impossible indépendamment du jour de la semaine, les jours fériés par exemple.

Jours de préparation

Nombre de jours avant qu'une livraison soit possible. Si vous le définissez à 3, un client qui commande un lundi ne pourra pas choisir de jour de livraison avant jeudi.

Cette option est redondante, mais plus simple, avec l'option "Date limite de livraison"

Jours affichés

Lorsque le client commande il choisit son jour de livraison dans une liste (même si c'est affiché sous forme de calendrier). Indiquez ici la longueur de cette liste. Attention si vous indiquez 30 jours la liste ne s'arretera pas à la fin du mois prochain mais aura 30 jours de livraison possible. Si vous ne livrez que le lundi, il y aura 30 lundi dans la liste.

Plage horaire

En plus du jour de livraison le client peut choisir un élément dans la liste que vous définissez ici. En général il s'agit de plage horaire (matin/midi/soir ou 9h/10h/11h...) mais vous pouvez mettre ce que vous voulez.

Laissez la liste vide si vous ne voulez pas utiliser cette fonctionnalité.
