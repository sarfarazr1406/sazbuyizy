Introduction

This module allows the customer to choose a delivery day.

You can export the orders and deliver products to prepare for a specific date.

If you want the requested delivery date may appear on your order and delivery. In this case a modification Prestashop files is required.

Module Installation

Installing from the administration panel

This is the simplest method, but it may not work in some cases. If this does not work, use the manual installation method presented below.

In the Modules tab, click to add a new module.

Click Browse to open a dialog box that allows you to search in your computer, select the file and confirm the dialog box. Finally, click Download this module.

The module will appear in your list of modules in the Module tab.

Click the Install button to install the module.

Install a module manually

Use this method if the manual installation does not work. It may seem tedious, a priori, for beginners, because it requires some knowledge of FTP, etc., but it is no more complex and perhaps more convenient because it works every time.

In addition, the expertise required for installation is pretty basic and it is something that all store managers must master.

To manually install a new PrestaShop module:

  1. Unzip the module archive file (.zip). This should create a new folder.
  2. Using your FTP client, place the folder in your PrestaShop folder /modules. Be careful to not upload this folder into the folder of another module (which can happen when you drag elements). Download the entire folder, not just the files in it.
  3. Go to your back office, in the tab "Modules".
  4. Locate the new module in the module list. You may have to scroll.
  5. Click the Install button to install the module.

Usage

  1. Define the possible delivery days in the module configuration page
  2. When a customer places an order, a calendar appears next to the selection of the carrier allowing him to choose a delivery date.
  3. The chosen date will appear in the order detail

Parameters

Manage by group

You can specify days for delivery globally for all clients or by groups of clients. It can be helpful if, for example, customers are classified by delivery area.

Enable for ...

This option only appears if "Manage by group" is enabled. Choose which groups of customers can choose a delivery day.

Required to order

Enable this option to force the client to choose a delivery day. He can not finalize its order otherwise.

Send delivery date to ...

When an order is created the module can send the requested delivery date by e-mail to an employee. Select "Nobody" to disable this feature.

Send delivery date to customer

When an order is created the module can send the requested delivery date to the customer.

Use separated mail

Appears only if "Send delivery date to customer" is enabled. The requested delivery date can be sent in the order confirmation or in a separate e-mail.

This email can be changed in the translation tool of Prestashop.

Show delivery date in orders list

This option adds a column "Delivery Date" in the "Commands" tab

Delivery days

If the "Manage by group" option is enabled, the following values ​​must be defined for each group.

Deliver

For each day of the week, choose if delivery is possible

Deadline for order

For each available day you must define the point at which it is no longer possible to order. For example, if to deliver an order on Tuesday it must be passed before Monday 2h PM, In front of Tuesday indicate 1 day(s) before at 14:00

Non delivery days

Select the days when the delivery is not possible regardless of day of the week, holidays, for example.

Days displayed

When a customer orders, he chooses a delivery day in a list (even if it is displayed as a calendar). Enter here the length of this list. Be careful, if you specify 30 days the list will not stop at the end of next month but will have 30 days of possible delivery. If you deliver only on Monday, there will be 30 Monday in the list.

Hourly ranges

In addition to the day of delivery, the customer can select an item in the list that you set here. Generally it's time slot (morning/evening or 9h/10h/11h ...) but you can put whatever you want.

Leave the list empty if you do not want to use this feature.

Frequently Asked Questions

Some customers don't receive the delivery date by mail, why ?

Maybe the mails have not been translated in the language of this customer.

In the /modules/deliverydays/mail, there is a folder by language ("fr" for french, "en" for english...) copy the "en" folder to "it" (for italian language) for example and translate the content of this new folder.

With recent versions of Prestashop you can translate the content of mails in Back Office > Tools > Translations > E-mail template translations .
